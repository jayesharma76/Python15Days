# Python 15 Days Journey

## Phase 1: The Pythonic Core (Days 1–5)
**Focus:** Mastering the "Pythonic" way to write clean, efficient code.

### Day 1: Advanced Data Structures & Comprehensions
- Go beyond lists: `defaultdict`, `Counter`, and `deque` from `collections`.
- List, Dict, and Set comprehensions with nested logic.

### Day 2: Functional Programming Tools
- Mastering `lambda`, `map`, `filter`, and `reduce`.
- Using `itertools` for efficient looping and `functools.partial`.

### Day 3: Decorators & Context Managers
- Understanding closures. Writing custom decorators (useful for Web Auth/Logging).
- Using `with` statements and creating custom context managers using `contextlib`.

### Day 4: Object-Oriented Python (Advanced)
- Dunder methods (`__str__`, `__call__`, `__getitem__`).
- Property decorators (`@property`), `@classmethod`, and `@staticmethod`.

### Day 5: Error Handling & Logging
- Custom Exception classes.
- Setting up the `logging` module (crucial for production web apps).

## Phase 2: Data Science Essentials (Days 6–10)
**Focus:** Vectorized operations and data manipulation.

### Day 6: NumPy & Vectorization
- Understanding the `ndarray` and why it beats Python lists.
- Broadcasting rules and boolean indexing.

### Day 7: Pandas (Deep Dive Part 1)
- DataFrames and Series. Reading messy CSVs/JSONs.
- Indexing with `.loc` and `.iloc`.

### Day 8: Pandas (Deep Dive Part 2)
- `groupby`, `pivot_table`, and merge/join operations.
- Handling missing data and time-series analysis.

### Day 9: Data Visualization Strategy
- Matplotlib for basic plotting; Seaborn for statistical visuals.
- Choosing the right chart for the right data insight.

### Day 10: Introduction to Scikit-Learn
- The Estimator API: `fit()` and `predict()`.
- Building a basic linear regression or classification pipeline.

## Phase 3: Web App Development & Deployment (Days 11–15)
**Focus:** Building APIs and understanding the modern web stack.

### Day 11: Virtual Environments & Dependency Management
- `venv`, `pip`, and `requirements.txt`.
- Understanding `poetry` or `pipenv` for professional workflows.

### Day 12: Flask/FastAPI (Routing & Logic)
- Setting up a basic API. Handling GET/POST requests.
- Request parsing and JSON responses.

### Day 13: Database Integration (SQLAlchemy/SQL)
- Connecting Python to a database using an ORM.
- Performing CRUD (Create, Read, Update, Delete) operations.

### Day 14: Asynchronous Programming (asyncio)
- Understanding `async`/`await`.
- Why it matters for high-performance web apps (FastAPI).

### Day 15: Testing & Deployment Basics
- Writing unit tests with `pytest`.
- Brief overview of Dockerizing a Python app for deployment.
